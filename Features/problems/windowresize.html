Hint

In this simple project, you will need only "p tag" to describe the width and height of the HTML page size of the window.

Requirement

In order to test your html, you need to set two p tag ids as "windowWidth" and "windowHeight".

System will test your answer according to below scenrio.
Scenario: Window width and height
When the browser open html file with system defined width and height
Then the current width and height of the window is shown
When browser is maximized, the changed size of the window width and height is shown.

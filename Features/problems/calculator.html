Hint

For layout interface, you can simply create your UI according to figure. Table layout and onclick function is used to getting the inputs.

Requirement

All numbers and operators(add, subtract,divide, multiply, equal,dot, clear) should be input button type except result output need to be input text type
In order to test your html, you need to set text input id and button id according to figure.
As it is case sensitive, you need to be careful of upper or lower case naming in button Id.

System will test your answer according to below scenrio.
Scenario : Calculator Function Flow
Then there should be blank in the result at the initial display state
When user input "2" and "3" to calculator to add
Then user get addition result "5"
Then user cleared the text
When user input "5" and "3" to calculator to subtract
Then user get subtraction result "2"
Then user cleared the text
When user input "2" and "4" to calculator to multiply
Then user get multiply result "8"
Then user cleared the text
When user input "6.8" and "4" to calculator to divide
Then user get division result "1.7"

Calculator Project

Create and develop simple "Calculator" using HTML, CSS and JavaScript. Your calculator may contain numbers (0-9), simple basic calculations (+, -, *, /, %, =) and result area to show the calculation answers. You should refer to detail specification about this project by clicking the show specification button. You can also try to write and run codes in the 'Code' area and see how your codes work in the 'Output' area. If your code is Ok, you can check your result by pressing get result button.
Please develop according to the requirement specification Scenario for testing.(Press Show Specification Button)
